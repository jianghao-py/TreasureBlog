### 键盘

<br>

- 键盘事件只能绑定给可以获取焦点的元素或document

<br>


| 事件名称 | 何时触发                                                       |
| -------- | -------------------------------------------------------------- |
| keydown  | 按下任意按键（不区分字母大小写，可以识别包括功能键内的所有键） |
| keypress | 除 Shift、Fn、CapsLock 外任意键被按住（区分字母大小写）        |
| keyup    | 释放任意按键（不区分字母大小写，可以识别包括功能键内的所有键） |  |




<br>

<br>

### e.keyCode

- 返回按键对应的ASCII码值
