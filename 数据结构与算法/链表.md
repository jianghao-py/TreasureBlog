

#链表

 1. 链表事宜接点的方式存储，属于链式存储
 2. 每个节点包含data域名，next域名：指向下一个节点
 3. **链表的各个节点不一定是连续存储**
 4. 链表有带头节点链表与不带头节点链表

###单链表实现

> 创建节点类
```java
class Node{
	public int num;
	public String name;
	public String nickname;
	public Node next; //指向下一个节点
	//构造器
	public Node(int Num,String Name, String Nickname) {
		this.num = Num;
		this.name = Name;
		this.nickname = Nickname;
	}
	@Override
	public String toString() {
		return "Node [num = " + num + ", name" + name + ",nickname = " + nickname + "]";
	}
}
```

> 显示链表
```java
        public void list() {
            	//判断链表是否为空
            	if(head.next == null) {
            		System.out.println("null");
            		return;
            	}
            	//head不能用，需要temp代替
            	Node temp = head.next;
            	while(true) {
            		//判断是否到链表最后
            		if(temp == null) {
            			break;
            		}
            		System.out.println(temp);
            		//链表节点后移，防止死循环
            		temp = temp.next;
            	}
            }
        }
```

> 添加节点（直接添加到链表尾部）
```java
 public void add(Node node) {
    	Node temp = head;
    	while(true) {
    		//找到链表的最后
    		if(temp.next == null) {
    			break;
    		}
    		//没有找到最后，后移temp
    		temp = temp.next;
    	}
    	//最后一个节点指向新添加的node
    	temp.next = node;
    }
```

> 添加节点（按照number插入到指定位置)
```java
//考虑编号顺序,按编号添加
    public void addByOrder(Node node) {
    	Node temp = head;
    	boolean flag = false;//flag标志添加的编号是否存在，默认为false
    	while(true) {
    		//找到链表的最后
    		if(temp.next == null) {
    			break;
    		}
    		//位置已经找到，停止寻找准备添加
    		if(temp.next.num > node.num) {
    			break;
    		//编号存在，不能添加
    		}else if(temp.next.num == node.num) {
    			flag = true;
    			break;
    		}
    		temp = temp.next;
    	}
    	if(flag) {
    		System.out.println("Can't add");
    	}else {
    		node.next = temp.next;
    		temp.next = node;
    	}
    }
```

> 修改节点
```java
 //修改节点
    public void change(Node newNode) {
    	if(head.next == null) {
    		System.out.println("链表为空");
    		return;
    	}
    	Node temp = head;
    	boolean flag = false;
    	while(true) {
    		//此时已经遍历完链表
    		if(temp == null) {
    			break;
    		}
    		//找到相同编号的节点，停止寻找
    		if(temp.num == newNode.num) {
    			flag = true;
    			break;
    		}
    		temp = temp.next;
    	}
    	if(flag) {
    		temp.name = newNode.name;
    		temp.nickname = newNode.nickname;
    	}else {
    		System.out.println("没有找到" + newNode);
    	}
    }
```

> 删除节点
```java
//删除节点
    public void delete(int num) {
    	Node temp = head;
    	boolean flag =false;
    	
    	while(true) {
    		if(temp.next == null) {
    			break;
    		}
    		//找到要删除的前一个节点，停止查找
    		if(temp.next.num == num) {
    			flag = true;
    			break;
    		}
    		temp = temp.next;
    	}
    	
    	if(flag) {
    		//此时为删除的前一个节点，设置前节点的next为下下个节点，此时被删除节点会被垃圾回收器回收
    		temp.next = temp.next.next;
    	}else {
    		System.out.println("没找到");
    	}
    }
```
